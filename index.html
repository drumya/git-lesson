• git clone path-to-repository - скачать репозиторий на локальный компьютер (с указанием ссылки)
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый файл под контроль Git (с указанием названия файла)
  • git commit -am "added new file index.html" - совершить, зафиксировать "совершенное действие"
  • git push - загрузить наш проект в Git репозиторий
  • git pull - скачать изменения на свой локальный компьютер
